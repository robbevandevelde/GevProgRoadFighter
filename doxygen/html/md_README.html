This is a project made by Thibaut Van Goethem for the course "gevorderd programmeren" The project is an implementation of the NES game roadfighter using SFML 2.4

Requirements

- SFML 2.4
- cmake 3.6 minimum
- a version of gcc/g++ or clang (tested with gcc 7.3.0 but most version should do)
- linux (preferably ubuntu), other operating systems have not been tested

how to run


command line:


- clone the project from github
- make new directory
- cd into that directory
- run cmake from the new directory onto the directory you cloned the project in
- run make in the directory you made
- the compiled version can now be found in the SFMLConversion map and is named GevProgRoadFighter

note 1: if you were to move the compiled version be sure to also move the resources directory with it
note 2: test made with the google framework can now also be run with the compiled version in the Tests directory

controls

left/right arrow key: move left/right
up/down arrow key: accelerate/decelerate
space: shoot (you can only shoot once in a certain time)
escape: pause game or exit at the end
